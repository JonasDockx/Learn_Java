# Established: intermediate Python programmer, new to Java

Jonas is at an **intermediate level in Python** and a beginner in JavaScript.
Java itself is new to him. Use **Python** as the comparison language when framing
new Java concepts — JavaScript analogies are not helpful and should be avoided.
Starting a junior Java developer role in a few months (see [[MISSION.md]]).

**Implications:** Skip programming primitives explained from zero (variables,
loops, functions as *concepts*) — he has these from Python. Teach Java's
*specifics* and the *deltas from Python*: static typing vs Python's dynamic
typing, the compile step vs Python's interpreter, Java's OO-by-default and
verbosity, the enterprise ecosystem. Pace can be brisk on fundamentals, slower
on genuinely new territory (the JVM, generics, Spring).

**First lesson taught:** 0001 — how Java runs (source → bytecode → JVM). Not yet
demonstrated as learned; awaiting evidence (quiz / running Hello.java) before
promoting toolchain terms to the glossary.
