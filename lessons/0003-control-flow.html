Control Flow

Conditionals and loops — concepts you own from Python, in Java's syntax, with two traps to learn now.

Mostly familiar ground. You already think in if, for, and while. So this lesson is deliberately fast on the ideas and careful on the differences: Java's braces and parentheses, its word-free operators, and one classic trap that bites every Python developer in their first Java week.

Conditionals

Python leans on indentation and the words and/or/not. Java uses ( ) around the condition, { } around the body, and symbol operators:

// Python                  // Java
// if age >= 18 and ok:    if (age >= 18 && ok) {
//     print("in")             System.out.println("in");
// elif age > 0:           } else if (age > 0) {
//     print("wait")           System.out.println("wait");
// else:                   } else {
//     print("no")             System.out.println("no");
//                         }

Trap #1 — comparing text with ==

In Python, name == "Jonas" compares the text. In Java, == on a String compares identity (are these the same object?), not the characters — so it often returns false even when the text matches. Rule for now: use == for primitives (int, boolean…) and .equals() for String and other objects: 
if (name.equals("Jonas")) { ... }   // ✔ compares the text
if (name == "Jonas")      { ... }   // ✗ compares object identity — avoid
We'll explain why when we reach objects and references. For now, just bank the rule.

Loops

Three you'll use. The for-each is closest to Python's for x in items:; the classic for gives you an index; while is the same idea everywhere.

int[] scores = {90, 80, 70};

// for-each  (like Python's: for s in scores)
for (int s : scores) {
    System.out.println(s);
}

// classic for  (gives you the index i)
for (int i = 0; i < scores.length; i++) {
    System.out.println(i + ": " + scores[i]);
}

// while
int n = 3;
while (n > 0) {
    System.out.println(n);
    n--;            // n-- means n = n - 1
}

Read the classic for as three parts separated by ;: start (int i = 0), keep going while (i < length), each step (i++). i++ adds one; n-- subtracts one.

Do this now

Write Grades.java that loops over scores and labels each:

public class Grades {
    public static void main(String[] args) {
        int[] scores = {95, 72, 58, 88};
        for (int s : scores) {
            if (s >= 90) {
                System.out.println(s + ": A");
            } else if (s >= 60) {
                System.out.println(s + ": pass");
            } else {
                System.out.println(s + ": fail");
            }
        }
    }
}
  1. Compile and run it. Confirm the labels match the scores.
  2. Rewrite the for-each as a classic for using the index i and scores[i] — same output.
  3. Add a String name = "Jonas"; and print a greeting only if (name.equals("Jonas")). Try it with == too and notice the difference.
