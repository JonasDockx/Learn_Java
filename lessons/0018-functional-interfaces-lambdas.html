Functional Interfaces & Lambdas

One idea: a lambda is a short way to implement an interface that has exactly one method.

Implementing an interface so far meant writing a whole class (like Lamp implements Switchable). When the interface has just one method, that's a lot of ceremony for one line of logic. A lambda is the shortcut — and it's the foundation of streams, next lesson.

The setup: a one-method interface

An interface with exactly one abstract method is called a functional interface:

interface Greeter {
    String greet(String name);     // one method — that makes it "functional"
}

The long way vs the lambda

To provide an implementation, you could write a full class. But if all you have is one method, Java lets you write just the method, inline, as a lambda:

// The whole class, just to hold one method:
class FriendlyGreeter implements Greeter {
    public String greet(String name) { return "Hi " + name; }
}
Greeter g1 = new FriendlyGreeter();

// The lambda — same thing, one line:
Greeter g2 = name -> "Hi " + name;

Both give you a Greeter whose greet returns "Hi " + name. You call them identically:

System.out.println(g2.greet("Jonas"));   // Hi Jonas

The key insight — read the arrow

name  ->  "Hi " + name
 ↑params  ↑arrow  ↑what it returns
Left of -> are the method's parameters; right of it is the result. Java already knows from the Greeter type that this lambda is a greet method, so you skip the class name, the method name, and the types. You're writing only the part that differs: the logic.

Python comparison

# Python                        // Java
greet = lambda name: "Hi " + name   Greeter greet = name -> "Hi " + name;
greet("Jonas")                      greet.greet("Jonas");

Nearly identical idea. The Java difference: a lambda must match a functional interface (here Greeter) — that interface is the lambda's "type". Java has no free-floating functions; a lambda is always an implementation of some one-method interface.

Multiple parameters

interface Adder { int add(int a, int b); }

Adder sum = (a, b) -> a + b;     // two params: wrap them in ( )
System.out.println(sum.add(2, 3));   // 5
