Part 3 — Small project (build with it)

A Gradebook that combines a HashMap, streams, lambdas, generics, and exception handling — every piece you've practised. One file, Gradebook.java.

Step by step

  1. Store grades. A HashMap<String, Integer> grades = new HashMap<>();. put in four students with scores (mix of pass/fail, pass = 60+).
  2. Count passers with a stream. grades.values().stream().filter(s -> s >= 60).count() — print it.
  3. List passers' names. grades.keySet().stream().filter(name -> grades.get(name) >= 60).toList() — print the list.
  4. Validate input with exceptions. Write static void addGrade(HashMap<String,Integer> m, String name, int score) that throws IllegalArgumentException if score is below 0 or above 100, otherwise puts it. Call it inside a try/catch with one bad score (e.g. 120) and confirm the program prints the error and keeps running.
  5. Commit on a branch and merge.

Why this is the real test

If you can build this without copying from earlier files, you've integrated the Advanced track: a keyed collection, generic types, two stream pipelines with lambdas, and exception handling that keeps the program alive — together.
