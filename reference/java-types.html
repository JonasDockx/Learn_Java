Types & Variables

The everyday types and the rules for declaring them. Print this.

Declaration shape

  type   name   =   value ;
  int    count  =   42    ;

The type is fixed for the variable's whole life. Re-assigning a different type = compile error.

Quote rules (easy to trip on)

  • Double quotes → String: "Jonas"
  • Single quotes → char: 'A'
  • boolean values are true / false — lowercase (Python uses True/False).

var — inferred, not dynamic

var count = 42;       // inferred int   — still fixed forever
var name  = "Jonas";  // inferred String
var x;                // ERROR — needs a value to infer from

Mental model

Python: the value carries the type, the name is a free label.
Java: the variable carries the type, stamped at declaration and never changed. var just lets the compiler fill in the stamp for you.

String joining

String name = "Jonas";
int count = 3;
System.out.println(name + " has " + count);   // + glues into text
