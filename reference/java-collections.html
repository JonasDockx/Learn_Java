ArrayList

The everyday resizable list. Print this.

Setup

import java.util.ArrayList;            // it's in java.util — must import

ArrayList<String> names = new ArrayList<>();   // a list of Strings, empty

<String> = the element type (what it holds). The empty <> on the right infers the same type.

Remember

Import it (java.util) · the <Type> fixes what it holds (static typing on contents) · methods not operators (.add/.get/.size).

HashMap

Key → value lookup. Python's dict.

Setup

import java.util.HashMap;

HashMap<String, Integer> ages = new HashMap<>();   // key type, value type

Two types in the <>: key first, value second. Values that are numbers use Integer (object form of int) because collections hold objects.
