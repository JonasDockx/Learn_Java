Condition in ( ), body in { }. else if is Java's elif.

Loops

// for-each  (Python: for s in scores)
for (int s : scores) { ... }

// classic for  — start ; keep-going ; step
for (int i = 0; i < scores.length; i++) { ... }

// while
while (n > 0) { n--; }

i++ = add 1 · n-- = subtract 1 · arr.length = size (no parentheses).

The String comparison rule

name.equals("Jonas")   // ✔ compares text — use for String/objects
name == "Jonas"        // ✗ compares identity — only for primitives
Primitives (int, double, boolean, char) → ==. Objects (String, …) → .equals().
