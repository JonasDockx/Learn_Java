One-time setup

git config --global user.name  "Your Name"

The everyday commands

CommandDoes
git initmake this folder a repo (once)
git statuswhat's changed / staged (use constantly)
git add <file>stage one file
git add .stage everything changed
git commit -m "msg"seal a save point
git log --onelinelist commit history
git diffshow unstaged changes line by line
git restore <file>discard uncommitted changes to a file

Good commit messages

Finish the sentence "If applied, this commit will…":
✔ Add Dog class demonstrating references
✗ stuff · ✗ fixed it · ✗ asdf
Present tense, specific, one logical change per commit.

.gitignore essentials (Java)

*.class      # bytecode — regenerated by javac
target/      # Maven output      build/   # Gradle output
.idea/ .vscode/ *.iml            # IDE noise
.DS_Store Thumbs.db              # OS clutter
*.key *.pem id_rsa .env          # SECRETS — never commit

Add .gitignore before your first git add. A leaked secret in history stays there.
