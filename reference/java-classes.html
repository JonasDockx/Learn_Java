Anatomy of a class

public class Dog {        // blueprint
    String name;          // field   (typed, declared up top)
    int age;              // field

    Dog(String name, int age) {   // constructor (named after the class)
        this.name = name;         // this = Python's self (implicit)
        this.age  = age;
    }

    String bark() {       // method  (typed return)
        return name + " woof";
    }
}

Reference rules

Dog b = a;        // copies the ARROW — a and b share one object
b.age = 4;        // visible through a too
a == c            // true only if a and c are the SAME object
a.equals(c)       // compares contents — IF the class defines equals()
null              // an arrow pointing at nothing

Run a class with a main

javac Dog.java     // compile
java Dog           // runs Dog's main method
